📘 Hướng dẫn Sử dụng API

Hệ thống cung cấp API để truy cập dữ liệu dân số Việt Nam. Bạn có thể sử dụng các endpoint dưới đây để lấy thông tin.

📌 1. Lấy Dữ liệu Dân Số Theo Năm

GET /api/users/population?year=2020

Mô tả: Lấy thông tin dân số của Việt Nam theo từng năm.

Tham số: year (tùy chọn, mặc định: 2021)

Ví dụ:

    $token = "Bearer your-token"
    $year = "2021"
    $url = `https://api.pplt-dev2.vitlab.site/api/users/population?year=$year`
        
    curl -H "Authorization: $token" "$url"

📌 2. Lấy Dữ liệu Dân Số Theo Tỉnh

GET /api/users/population?province=Vĩnh%20Long

Mô tả: Lấy thông tin dân số Tỉnh theo từng năm.

Tham số: province (tùy chọn, mặc định: Vĩnh%20Long)

Ví dụ:

    $token = "Bearer your-token"
    $province = "Vĩnh%20Long"
    $url = `https://api.vnpop.thonh.site/api/users/population?province=$province`
        
    curl -H "Authorization: $token" "$url"

📌 3. Đăng ký Người Dùng(Deploying)

POST /api/users/register

Body (JSON):

{
  "full_name": "Nguyễn Văn A",
  "email": "user@example.com",
  "password": "123456"
}

📌 4. Đăng nhập Người Dùng (Deploying)

POST /api/users/login

Body (JSON):

{
  "email": "user@example.com",
  "password": "123456"
}

📌 5. Lấy Thông Tin Người Dùng (Deploying)

GET /api/users/profile

Mô tả: Lấy thông tin của người dùng đã đăng nhập.

Header: Authorization: Bearer [TOKEN]
